# Time Fighter

Simple Android project with a multi-purpose test.

## Getting Started

1\. Run the test task:

```shell
$ ./gradlew test
```

2\. Run the connectedAndroidTest task: 

```shell
$ ./gradlew connectedAndroidTest
```
